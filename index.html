Q.1 difference between flex row and column?

Ans.1 'flex row' is the direction of items in the flex container to be aligned horizontally in the webpage. It this is by default setting of display flex.

whereas

'flex column' is the direction of flex items in the flex container to be aligned vertically in the webpage.

Q.2 Explain inline,internal and external stylesheet?

Ans.2 There are three ways by which we can add styling to our html webpage. These are 'inline','internal', and 'external stylsheet'

'inline' this type of styling is to be done inside the opening tag body of the element to be styled, in this type "style" is used as an attribute inside which css styling properties is to be written.

'internal' this type of styling is to be coded inside the head tag of the webpage. In this type of styling "style" is used as the element inside which the css properties is to be written down corresponding to the selector.

'external stylesheet' this type of styling is to be coded in a seperate file(.css), and should be linked inside the corresponding html page inside the head tag using the link tag.

Q.3 justify-content allows you to do what?

Ans.3 'justify-content' is the css property used for flex container in order to arrange/accomodate/ditribute the horizontal space present in the flex container among the flex items/content

Q.4 difference between absolute and relative positioning?

Ans.4 'absolute positioning' is the method to position the element in the webpage. In absolute position, all four side parameters(top,right,bottom,left) is calculated with respect to the top left corner of the parent element.

'relative positioning' method is used when we want to position the element in the webpage, with respect/relative to its static/default position.

Q.5 what is grid-template-columns used for?

Ans.5 'grid-template-columns' is an essential property for the grid container, it is used to define the no. of columns and their width inside the grid.

Q.6 what is z-index in css?

Ans.6 'z-index' is the special property of css, it is used to give/arrange the elements in 3d mode/along the z-axis/in the form of stack. The basic requirement for implementation of z-index is that the elements should be positioned in any of the way except(static).

Q.7 what is the difference between padding and margin?

Ans.7 1. 'margin' presents/applies outside the border/body of the element/box. Whereas, 'padding' is given around the main content inside the border/body of box/element.

2. The size parameters of the margin applied does not calculated along with the actual size of the box/element. Whereas, the parameters of the padding applied is included in the actual size of box/element.
